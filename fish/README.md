# fish

i made basically no changes to fish, the only plugins i use are:
- [tide (for the prompt)](https://github.com/IlanCosman/tide)
- [replay (qol)](https://github.com/jorgebucaran/replay.fish)
- and [z (for directory jumping)](https://github.com/jethrokuan/z)
and [fisher](https://github.com/jorgebucaran/fisher) for plugins, ofc

---

my tide config is:
```
tide configure --auto --style=Lean --prompt_colors='16 colors' --show_time=No --lean_prompt_height='One line' --prompt_spacing=Sparse --icons='Few icons' --transient=No
```
